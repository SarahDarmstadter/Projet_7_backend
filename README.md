# Getting Started 
Create .env 
Paste following code :
    PORT= your port 
    SECRET_KEY = your secret key 
    HOST = your IP adress
    USER =  your database user
    PASSWORD = your database password
    DB = database_groupomania

### LAUNCH SERVER 
nodemon server
